# 🌍 TrendExplore

Global conversation explorer – Cross cultural trending analytics dashboard.

## 🚀 Tech Stack
Frontend: React + Tailwind CSS  
Backend: FastAPI + MongoDB  
Charts: Recharts  

## 📊 Features
- View trending topics by country
- Compare trends across countries
- Real-time chat system
- Cross-cultural analytics dashboard
